# supply-chain-analytics
Agile supply chain analytics sprint project

## Project Description
This project analyzes supply delivery history data using Agile practices, GitHub collaboration, and Python data processing.

## Sprint Goal
The sprint goal is to load, clean, and prepare the supply delivery dataset for analysis.

## Team Workflow
- Each member works on a separate feature branch
- All work is submitted through Pull Requests
- Pull Requests are reviewed before merging
- Taiga stories are linked with GitHub commits and PRs
